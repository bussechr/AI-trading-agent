# Orchestration Golden Captures

Phase 0 baseline packs generated by `tools/capture_baseline_pack.py` live under this directory.

Each capture directory contains:

- raw `/v2/ready`, `/v2/state`, and `/v2/decision-snapshots` payloads
- a normalized baseline summary
- capture metadata, version metadata, and environment facts

These files are golden parity seeds for later shadow and replay phases. They are intentionally additive and should not be rewritten by unrelated tests.
